# Nike Financial Data Analysis

## Project Overview
Analysis of Nike's financial performance using data extracted from 10-K annual reports to evaluate profitability, financial health, trends, and future scenarios.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit

## Analysis Performed
- Financial statement analysis
- Ratio analysis
- Revenue trend analysis
- Forecasting
- Scenario analysis

## Key Insights

- Nike's revenue remained stable from 2023 to 2024 but declined by approximately 10% in 2025, indicating weaker sales performance.
- Gross margin decreased from 44.56% in 2024 to 42.73% in 2025, highlighting pressure on profitability.
- Net income declined significantly in 2025, reducing net margin from 11.10% to 6.95%.
- Return on Assets (ROA) decreased from 14.96% in 2024 to 8.80% in 2025, showing lower efficiency in generating profit from assets.
- Scenario analysis showed that revenue recovery could improve profitability, while continued decline may negatively impact future performance.
- Forecasting analysis estimated future financial outcomes using revenue growth assumptions and historical financial ratios.
